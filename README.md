# Graphs Project

## Overview

This project explores graph data structures and traversal algorithms, including Breadth-First Search (BFS) and Depth-First Search (DFS).

The full written report can be found here:

* `docs/report.md`

---

## Code Structure

* `code/graph.py` — Graph implementation (adjacency list)
* `code/search.py` — BFS and DFS algorithms
* `code/test_graph.py` — Test cases
* `code/main.py` — Runs all tests and prints results

---

## Running the Code

Run all tests with:

```bash
python code/main.py
```

This will execute all test cases and display whether they pass or fail.

---

## Notes

* The graph is implemented as a **directed, unweighted graph**
* The structure allows easy extension to **weighted graphs**
* BFS is used for shortest path (unweighted)
* DFS is used for full exploration
